# ShipFlip 🚀↔️

> Marketplace for freshly-built MVPs — from builders to operators.

**Ship it, then flip it.**

Vibe-coded services need operators, not more code.
ShipFlip connects developers who build fast with entrepreneurs who grow fast.

## Concept

- **Sellers**: Developers who ship MVPs quickly using AI/vibe-coding but don't want to operate
- **Buyers**: Business operators who can't code but know how to grow and monetize
- **Focus**: Pre-revenue to early-revenue services (₩0 ~ ₩10M ARR)
